# portal-home-guide

## 项目简介

`portal-home-guide` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为一组静态页面的集中存放和版本管理空间。你可以在此浏览、下载或参考这些页面的实现方式。

## 目录说明

```
/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面所需的静态资源（CSS、JS、图片等）
└── README.md       # 本文件
```

- `pages/`：每个子文件夹或文件对应一个独立页面，命名清晰以方便识别。
- `assets/`：按需组织，资源文件与页面分离，便于维护和复用。

## 页面归档说明

本仓库中的 HTML 页面为独立归档，不依赖外部服务或特定域名。每个页面均为自包含或通过仓库内资源引用，可离线或静态部署使用。

- 页面内容聚焦于通用功能或展示，不涉及具体商业、营销信息。
- 所有页面均为静态 HTML，可直接在浏览器中打开运行。

## 维护说明

- 欢迎提交 pull request 添加新的页面归档，或对现有页面进行改进。
- 请保持页面独立性，避免引入外部不可控资源。
- 提交前请确保页面在主流浏览器中正常显示。
- 仓库维护者会定期检查页面兼容性和内容合理性。
